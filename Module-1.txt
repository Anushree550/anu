1) What is Software?

-> Software is a set of instructions, programs, or applications that tell a computer how to perform specific tasks. It is typically stored on a computer's hard drive    and can be accessed and used by the computer's operating system.

3) What is programing?

-> Programming is the process of creating instructions for a computer or other device to carry out specific tasks. It involves writing code using a programming    language, such as Java, C++, Python, and others. Programming is used to develop software applications, websites, games, and much more.
